HidSharp Sample

HIDSharp is a multiplatform C# library for USB HID devices written by James F. Bellinger. It is available at https://github.com/IntergatedCircuits/HidSharp

The HidSharp sample for X-keys can be found in the PI Engineering SDK\HidSharp Console folder. It is written using VS2022.
